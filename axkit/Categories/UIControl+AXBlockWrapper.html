– ax_removeEventHandlersForControlEvents:

Removes all control event blocks associated with the given mask of control events.

- (void)ax_removeEventHandlersForControlEvents:(UIControlEvents)controlEvents

Parameters

controlEvents

A bitmask specifying the control events for which the block will be removed.

Discussion

Do note that, like @c UIControl, this method will not decompose control events; thus, only a handler matching an exact given bitmask will be removed.
